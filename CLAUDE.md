## gstack
Use /browse from gstack for all web browsing. Never use mcp__claude-in-chrome__* tools.

> Note: in this sandboxed remote environment, `/browse` cannot launch because
> egress to `cdn.playwright.dev` (Playwright's Chromium download) is blocked
> by network policy and the pre-installed Chromium build here is a different,
> incompatible revision. Fall back to WebFetch/WebSearch for web browsing in
> this environment until that's resolved.
